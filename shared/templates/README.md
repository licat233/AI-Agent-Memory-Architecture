# Template Profiles

Choose one active template profile per installed template folder.

| Folder | Profile |
| --- | --- |
| `armor/plain/` | Runtime-neutral ARMOR templates |
| `armor/obsidian-core/` | ARMOR templates using official `{{date}}` and `{{title}}` variables |
| `armor/templater/` | ARMOR templates using `<% tp.* %>` |
| `pama/plain/` | Runtime-neutral PAMA templates |
| `pama/obsidian-core/` | PAMA templates using official variables |
| `pama/templater/` | PAMA templates using `<% tp.* %>` |

Read `TEMPLATE_AUTOMATION_GUIDE.md` before installation.
